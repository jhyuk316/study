## 4. WAS

- Web Application Server
- WAS = Web Server + Web Container
- DB 조회나 다양한 로직 처리를 요구하는 동적인 컨텐츠를 제공하기 위해 만들어진 Application Server
- **웹 컨테이너(Web Container)** 혹은 **서블릿 컨테이너(Servlet Container)** - JSP, Servlet을 실행시킬 수 있는 소프트웨어

- Web Server vs WAS
  - Web Server
    - 정적인 컨덴츠
  - WAS
    - 동적인 컨덴츠
    - Tomcat
